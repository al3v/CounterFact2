# ADL SAE Controlled QA Experiment

The project is about checking how small changes in a factual question can affect the answer of a language model.

## Goal of the project


Can surface/tokenization changes in a question make the model switch between a correct and wrong factual answer?

And if that happens:

Can we also see this change inside the model representations?
